Описание

Разработан пакет VectorMath, содержащий функции для выполнения основных операций над векторами:

  • VecAdd — сложение
  • VecSub — вычитание
  • VecDot — скалярное произведение
  • VecCross — векторное произведение (3D)
  • VecNorm — длина вектора
  • VecAngle — угол между векторами
  • VecScalar — умножение на скаляр

Каждая функция имеет три режима работы:

  • ind = 0 — подробное решение с пояснениями
  • ind = 1 — только результат
  • ind = 2 — графическая визуализация

Код

(* Загрузка пакета *) Needs["VectorMath`"] (* Сложение векторов с подробным выводом *) VecAdd[{1, 2}, {3, 4}, 0] (* Длина вектора с визуализацией *) VecNorm[{3, 4}, 2] (* Угол между векторами *) VecAngle[{1, 0}, {0, 1}, 0]

Результат

Сложение векторов
Вектор 1: {1, 2}
Вектор 2: {3, 4}
Результат: {4, 6}
