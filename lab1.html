Описание

Создана интерактивная 3D-модель, отображающая все возможные RGB-цвета как трёхмерный шар в пространстве.

Математическая модель: Каждая точка имеет координаты (R, G, B), где R, G, B ∈ [0, 255]. Центр RGB-куба находится в точке (127.5, 127.5, 127.5) — это белый цвет.

Управление:

  • шаг (step): определяет количество отображаемых цветов
  • размер шариков (size): визуальная плотность модели

При step = 32 получается 729 цветов. При step = 16 — 4913 цветов.

Код

(* Генерация точек *) points = Flatten[Table[{r, g, b}, {r, 0, 255, step}, {g, 0, 255, step}, {b, 0, 255, step}], 2]; (* Сдвиг к центру *) center = {127.5, 127.5, 127.5}; shiftedPoints = # - center & /@ points; (* Визуализация *) Graphics3D[ Table[ { RGBColor[points[[i, 1]]/255, points[[i, 2]]/255, points[[i, 3]]/255], Sphere[shiftedPoints[[i]], size] }, {i, Length[points]} ], PlotRange -> {{-130, 130}, {-130, 130}, {-130, 130}}, Axes -> True, AxesLabel -> {"R", "G", "B"}, ImageSize -> {450, 400} ]
