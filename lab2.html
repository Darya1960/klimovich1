Описание

Разработано приложение для поиска информации о фильмах через OMDB API. Функция SearchMovies находит информацию по названию:

  • Название фильма
  • Год выпуска
  • Рейтинг IMDB
  • Актёры
  • Сюжет
  • Постер

Код

(* Поиск фильма *) SearchMovies[title_String] := Module[ {url, data}, url = "http://www.omdbapi.com/?apikey=144fcc36&t=" <> StringReplace[title, " " -> "+"]; data = ImportString[URLRead[HTTPRequest[url, <|"Method" -> "GET"|>], "Body"], "RawJSON"]; { "Название" -> data["Title"], "Год" -> data["Year"], "Рейтинг IMDB" -> data["imdbRating"], "Актёры" -> data["Actors"], "Сюжет" -> data["Plot"], "Постер" -> data["Poster"] } ] (* Пример использования *) movie = SearchMovies["The Matrix"]
